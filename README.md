# Doctorsid's Bosses Mod

## Introduction
Welcome to the Doctorsid's Bosses mod! This mod introduces new bosses to enhance your gaming experience. It is designed to challenge players with unique abilities and gameplay mechanics.

## Features
- **Multiple Bosses**: Fight against various bosses each with unique abilities.
- **Challenging Gameplay**: Each boss presents its own set of challenges to keep players engaged.
- **Customizable Options**: Players can adjust settings to tailor the difficulty level.

## Installation Instructions
1. **Download the mod**: Obtain the latest version of the Doctorsid's Bosses mod from the official repository.
2. **Install the mod**: Place the downloaded files in your game's mod directory.
3. **Launch the Game**: Start the game and navigate to the mod menu to activate Doctorsid's Bosses.

## The Yeti Boss
### Overview
The Yeti is one of the fiercest bosses introduced in this mod. 

### Abilities
- **Frost Breath**: The Yeti can unleash a freezing breath that slows down players.
- **Snowstorm**: Calls forth a snowstorm that obscures vision and hinders movement.

### Strategy
- **Use Fire-Based Attacks**: The Yeti is vulnerable to fire, so fire-based weapons are highly effective.
- **Keep Moving**: Avoid staying in one spot to evade the Yeti's Frost Breath and Snowstorm attacks.  

Get ready to take on the Yeti and other bosses in Doctorsid's Bosses mod!
